# Tech-Stack: NextGen RTP-Generator

## 📋 Überblick
**Projekttyp**: CLI-Tool/Server-Tool
**Primärer Zweck**: Erstellen eines voll konfigurierbaren RTP-Streams für Test-Zwecke (gezielt Edge-Cases), mit möglichkeit aktiv nicht-RFC-konforme Streams zu erzeugen. 
**Repository**: https://github.com/dkuehnl/RTP-Gen-NG
